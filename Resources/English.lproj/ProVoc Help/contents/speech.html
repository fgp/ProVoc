Speech synthesizer

The speech synthesizer can be used to speak any English word during training. To enable the speech synthesizer, select the option Use speech synthesizer for English in the training parameters.

Note

The speech synthesizer only supports English.

Hint

To start speaking any text in ProVoc, hold down Control (⌃) and click (or right-click) on the corresponding item and select Start Speaking. You can also start speaking the selected words in the list.
