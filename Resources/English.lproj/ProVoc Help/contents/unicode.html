Unicode support

ProVoc supports unicode, meaning that you can enter words with any alphabet in any writing direction.

To choose different input methods for specific alphabets (i.e. to enter Greek, Japanese, Hebrew, etc.):

  • Select System Preferences > International > Input Menu
  • Check the desired input methods corresponding to the alphabet you are using
  • Select the "Show input menu in menu bar"
  • In ProVoc, select the input method from the menu bar (or use the shortcut set in the System Preferences) to access to the different alphabets

To select the writing directions for the different languages:

  • Select ProVoc > Preferences > Fonts
  • Choose the desired writing directions in the section Writing Directions

Note

The System Preferences provide a keyboard viewer (accessible from the input menu) to help you locate any special character on your keyboard.
