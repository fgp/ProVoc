Sharing vocabulary files

Vocabulary files can be shared among ProVoc users.

  • To download published vocabulary files, select Share > Download ProVoc Documents.
  • To make your own vocabulary file available, select Share > Submit Document.

Note

To update a published vocabulary file, simply submit the file again with the same title and author.
