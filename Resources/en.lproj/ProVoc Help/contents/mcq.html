Multiple choice questions

To select an answer from the proposed list:

  • Click with the mouse
  • Use the Up and Down arrow keys
  • Press the key 1, 2, 3, etc.

Tip: double-click an answer to directly verify it.

If you don't want the choices to be displayed immediately, check the option Delayed choices of the corresponding training mode.
