Managing languages

  • To select the two languages of the vocabulary file, use the popup menus in the tab Editing.
  • To create a new language, select ProVoc > Preferences and click the + button under the language list. To change the name of a language, double-click its name in this list.

During the training, answers are accepted as correct depending on the language preference settings:

  • Case: if not checked, the case is ignored (lowercase, uppercase)
  • Accents: if not checked, the accents are ignored
  • Punctuation: if not checked, punctuation signs or symbols are ignored
  • Spaces: if not checked, whitespace characters are ignored
  • Facultative Words: words which can be omitted in the answer (such as determinents), separated by commas.

Note that text in parentheses "(...)" can always be omitted in the answer.
